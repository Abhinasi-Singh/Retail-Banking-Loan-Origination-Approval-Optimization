# Stakeholder Analysis

This folder contains stakeholder-related artifacts for the Retail Banking Loan Origination & Approval Optimization project.

## Planned Deliverables

- Stakeholder Register
- Power/Interest Matrix
- RACI Matrix
- Stakeholder Communication Notes

## Purpose

The purpose of this section is to identify the individuals, teams, and groups impacted by the project and define how they should be managed throughout the project lifecycle.
